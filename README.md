# Temperature Converter

A simple, beginner-friendly C++ program that converts temperatures between **Celsius**, **Fahrenheit**, and **Kelvin**.

Perfect for learning basic C++ input/output, functions, and control structures!

## Features
- Convert from Celsius to Fahrenheit and Kelvin
- Convert from Fahrenheit to Celsius and Kelvin
- Convert from Kelvin to Celsius and Fahrenheit
- User-friendly menu interface
- Handles invalid inputs gracefully

## How to Use / "Install"

No complex installation needed — just a C++ compiler!

### Option 1: Quickest way (Download & Run)

1. Go to the green **Code** button (top right) → **Download ZIP**
2. Extract the ZIP file to any folder
3. Open a terminal/command prompt in that folder  
   (or just navigate to it using `cd path/to/folder`)

4. Compile and run:

   **On Linux, macOS, or Windows (with g++ installed):**
   ```bash
   g++ main.cpp -o converter
   ./converter          # Linux/macOS
   # or
   converter.exe        # Windows
